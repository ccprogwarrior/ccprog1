# Square Area
Write a program that solves for the area of a square given its side length user input

Area of square = side * side

<details>
<summary>Explanation</summary>
<br>
</details>


<details>
<summary>Answer</summary>
<br>

``` c
#include<stdio.h>
int main(){
	float side, area;
	scanf("%f", &side);
	area = side * side;
	printf("Area %f", area);
	return 0;
}
```
