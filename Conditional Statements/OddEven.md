# Odd or even
Write a program that determines if an input is odd or even

<summary>Explanation</summary>
<br>
</details>


<details>
<summary>Answer</summary>
<br>

``` c
#include<stdio.h>
int main(){
	int input;
	if(input%2==0){
		printf("Even");
	} else {
		printf("Odd");
	}
	return 0;
}
```
