# Higher Problem
Write a function that determines which of the 2 user input is higher

<summary>Explanation</summary>

</details>


<details>
<summary>Answer</summary>
<br>

``` c
int getHigher(int val1, int val2){
	int highervalue;
	if(val1> val2)
		highervalue = val1;
	else
		highervalue = val2;
	return highervalue;
}
```
