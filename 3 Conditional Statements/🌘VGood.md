# VGood Problem
Write a program that will display ‘V.Good!’ and ‘Keep it up!’ if the grade uof a student is 4.0.

<summary>Explanation</summary>
<br>
</details>


<details>
<summary>Answer</summary>
<br>

``` c
#include<stdio.h>
int main(){
	float grade;
	scanf("%f", &grade);
	if(grade==4.0){
		printf("'V.Good!' and 'Keep it up!'");
	}
	return 0;
}
```
