# isOdd Problem
Write a function that determines if a number is odd and uses the isEven function

<details>
<summary>Explanation</summary>
<br>
</details>


<details>
<summary>Answer</summary>
<br>

``` c
int 
isEven(int val){
	return val % 2 ==0;
}

int
isOdd(int val){
	return !isEven;
}
```
